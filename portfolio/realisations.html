Mes Réalisations

Site marchand Turbo Compétition (STAGE)

BTS SIO - 2024 - réalisé seul

Dans le cadre d’un stage de BTS SIO au sein du lycée Parc des Loges à Evry, j’ai eu l’opportunité de réaliser mon stage au sein d’une boutique de pièces automobiles, Turbo Compétition, à Lisses.
Durant 7 semaines, j’ai travaillé sur la mise en place d’un site marchand en répondant aux besoins de mon tuteur de stage tout en m’adaptant à l’environnement de l’entreprise et ses enjeux.
L’objectif principal de mon stage a été de mettre en application mes compétences acquises durant la 1ère année de BTS SIO dans un contexte spécifique.
Afin de pouvoir me faciliter la réalisation du site, j'ai utilisé un CMS qui est WordPress, avec l'extension WooCommerce pour le cas d'une gestion de boutique, grâce à l'hébergement assuré par IONOS.

Projet Xencelabs : Tablette à Stylet

BTS SIO - 2023/2024 - réalisé en binôme, puis seul

Il s'agit du premier projet de l'année en BTS SIO 1ère année permettant de mettre en application tout ce que nous avions appris lors du 1er semestre en développement web. L'objectif était de créer un site marchand avec les langages HTML, CSS et JavaScript sur l'éditeur de code Visual Studio Code et le thème qui nous avait été dédié était les tablettes à stylet. Nous avions tout d'abord réalisé une maquette de notre page d'accueil sur Canva, puis nous avions réalisé un cahier des charges pour noter tous nos besoins et idées pour le site marchand que ce soit les fonctionnalités, les produits à inclure, les codes couleurs...
Nous nous sommes organisés en se repartissant les taches sur Trello, en ayant pris aussi en compte la méthode Agile afin de pouvoir finir ce projet dans les temps imparti.
Ce projet nous a permis de solidifier nos connaissances en développement web sur des langages tel que HTML, CSS et JavaScript.
Ensuite, lors de la deuxième partie du projet qui a été réalisé seul, j'ai utilisé le langage PHP afin de dynamiser mon site avec une base de donnée (MySQL).
Cette fois, l'objectif était de dynamiser le site marchand qui n'était qu'au départ qu'un site vitrine grâce à la connexion du site avec une base de donnée avec PHP.
Je devais d'abord connecter mon site avec une base de donnée créé sur PHP My Admin, puis j'ai réalisé la fonction de connexion et d'inscription d'un utilisateur dans la base de donnée à travers un formulaire de connexion. Ensuite je devais inclure tous les produits dans la base de donnée et enfin je devais réalisé un panier fonctionnel.
